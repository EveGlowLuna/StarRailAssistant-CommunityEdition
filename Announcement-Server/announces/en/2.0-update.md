## Version 2.0 Update

### Main Updates:

- Added manual update check button
- Added restart console button
- Currency Wars now supports continuing progress. #67
- Support for custom keybindings. #67
- Automatic detection and creation of desktop shortcuts
- Customizable background images
- Language switching support (requires software restart), some translations may be incomplete, PRs are welcome

### Feature Adjustments:

- Currency Wars now requires entering the Trailblazer name (account name), cannot continue without it. #67
